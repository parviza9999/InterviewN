# Fraud Detection GCP Capstone

## Project Overview

This capstone project develops an end-to-end fraud detection system using machine learning and Google Cloud Platform. The system will identify suspicious financial transactions using automated data preprocessing, anomaly detection, supervised machine learning, model evaluation, and cloud deployment.

## Dataset Selected

The project will use only one dataset:

**Option 1: Credit Card Fraud Detection Dataset**

This dataset contains anonymized European credit card transactions and is highly imbalanced. It is appropriate for fraud detection, anomaly detection, class imbalance analysis, supervised model comparison, and model evaluation using precision, recall, F1-score, ROC-AUC, PR-AUC, and confusion matrix analysis.

Option 2, the Bank Account Fraud / BAF Suite, was reviewed but excluded from the final project scope. The team decided to keep the capstone focused on one complete end-to-end fraud detection solution using Option 1 only.

## Project Phases

1. Phase 1: Proposal, dataset selection, GitHub setup, and project planning
2. Phase 2: Data loading, exploratory data analysis, and preprocessing pipeline
3. Phase 3: Baseline model and Autoencoder anomaly detection
4. Phase 4: XGBoost supervised model comparison
5. Phase 5: GCP deployment using Cloud Run and/or Firebase Hosting
6. Phase 6: Final report, validation, and presentation
